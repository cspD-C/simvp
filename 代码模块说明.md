## 🔧 核心模块详解

### 1. **main.py** - 主入口文件

**作用**: 程序的启动入口,负责参数解析和流程控制

**主要功能**:
- `create_parser()`: 定义所有命令行参数
  - 基础设置: GPU、随机种子
  - 数据集参数: 批次大小、数据路径
  - 模型参数: 隐藏维度、网络层数
  - 训练参数: 训练轮数、学习率

**执行流程**:
1. 解析命令行参数
2. 创建 Exp 实验对象
3. 执行训练 (`exp.train()`)
4. 执行测试 (`exp.test()`)

**使用示例**:
```bash
python main.py --epochs 1000 --lr 0.001 --dataname mmnist
```

---

### 2. **modules.py** - 基础构建模块

**作用**: 定义 SimVP 模型的所有基础卷积模块

#### 模块1: `BasicConv2d` - 基础卷积层
- **功能**: 提供基础的 2D 卷积操作
- **特点**:
  - 支持普通卷积和转置卷积(上采样)
  - 可选 GroupNorm 归一化
  - 可选 LeakyReLU 激活函数

#### 模块2: `ConvSC` - 空间卷积模块
- **功能**: SimVP 的核心空间卷积单元
- **特点**:
  - 固定使用 3x3 卷积核
  - stride=1: 保持分辨率
  - stride=2: 下采样(编码器)或上采样(解码器)

#### 模块3: `GroupConv2d` - 分组卷积层
- **功能**: 使用分组卷积减少参数量
- **优势**: 参数量 = 原始参数量 / groups
- **应用**: 用于 Inception 模块

#### 模块4: `Inception` - 多尺度特征提取
- **功能**: 同时使用多个不同尺度的卷积核
- **卷积核尺寸**: 3x3, 5x5, 7x7, 11x11
- **关键**: 捕获不同感受野的时空特征

---

### 3. **model.py** - SimVP 模型架构

**作用**: 定义完整的 SimVP 视频预测模型

#### 组件1: `Encoder` - 编码器
- **功能**: 将输入视频帧编码为低维潜在表示
- **输入**: `[B*T, C, H, W]` - 所有帧
- **输出**: 
  - `latent`: 编码后的潜在表示
  - `enc1`: 第一层特征(用于跳跃连接)
- **方法**: 多层卷积逐步下采样

#### 组件2: `Decoder` - 解码器
- **功能**: 将潜在表示解码为预测的视频帧
- **输入**: 
  - `hid`: 来自 Mid_Xnet 的潜在表示
  - `enc1`: 来自编码器的跳跃连接特征
- **输出**: `[B*T, C, H, W]` - 预测的视频帧
- **方法**: 转置卷积逐步上采样

#### 组件3: `Mid_Xnet` - 时间建模核心 ⭐
- **功能**: 在潜在空间进行时间序列建模
- **架构**: U-Net 风格的编码器-解码器
- **关键**: 
  - 使用 Inception 模块进行多尺度时间特征提取
  - 编码器: 逐步压缩时间信息
  - 解码器: 逐步恢复时间信息
  - 跳跃连接: 保留不同层次的时间特征
- **核心作用**: 学习视频中的时间动态和运动模式

#### 完整模型: `SimVP`
- **组合**: Encoder + Mid_Xnet + Decoder
- **流程**:
  1. Encoder: 输入帧 → 潜在表示
  2. Mid_Xnet: 时间建模
  3. Decoder: 潜在表示 → 预测帧
- **输入**: `[B, T, C, H, W]` - 输入视频序列
- **输出**: `[B, T, C, H, W]` - 预测的未来视频序列

---

### 4. **exp.py** - 实验管理

**作用**: 管理整个训练、验证、测试流程

**主要类**: `Exp`

**核心方法**:

1. **`__init__()`** - 初始化
   - 设置设备(GPU/CPU)
   - 加载数据
   - 构建模型
   - 创建优化器

2. **`train()`** - 训练循环
   - 遍历所有 epoch
   - 前向传播 → 计算损失 → 反向传播
   - 定期验证和保存模型
   - 使用 OneCycleLR 学习率调度

3. **`vali()`** - 验证
   - 在验证集上评估模型
   - 计算 MSE, MAE, SSIM, PSNR 指标
   - 返回验证损失

4. **`test()`** - 测试
   - 在测试集上生成预测
   - 保存预测结果为 `.npy` 文件
   - 计算最终性能指标

**保存的文件**:
- `checkpoint.pth`: 最佳模型权重
- `inputs.npy`: 输入序列
- `trues.npy`: 真实未来帧
- `preds.npy`: 预测未来帧

---

### 5. **utils.py** - 工具函数

**作用**: 提供训练过程中的辅助函数

**函数列表**:

1. **`set_seed(seed)`**
   - 设置随机种子,确保实验可复现
   - 影响: Python random, NumPy, PyTorch, cuDNN

2. **`print_log(message)`**
   - 同时打印到控制台和日志文件

3. **`output_namespace(namespace)`**
   - 格式化输出配置参数

4. **`check_dir(path)`**
   - 检查并创建目录

---

### 6. **visualize.py** - 可视化脚本

**作用**: 从保存的 `.npy` 文件生成可视化图像

**主要功能**:

1. **`load_data()`** - 加载预测结果
2. **`visualize_sample()`** - 可视化单个样本
   - 展示输入序列、真实帧、预测帧
3. **`plot_errors()`** - 绘制误差曲线
   - MSE 和 MAE 随时间步的变化

**使用方法**:
```bash
python visualize.py
```

**生成的文件**:
- `sample_1.png` ~ `sample_5.png`: 样本可视化
- `errors.png`: 误差分析图

## 📊 数据流程图

```
输入视频序列 [B, T_in, C, H, W]
         ↓
    Encoder (空间编码)
         ↓
  潜在表示 [B, T_in, C', H', W']
         ↓
    Mid_Xnet (时间建模) ⭐ 核心
         ↓
  建模后潜在表示 [B, T_out, C', H', W']
         ↓
    Decoder (空间解码)
         ↓

```

## 🔑 关键参数说明

### 模型参数

| 参数 | 说明 | 默认值 | 推荐范围 |
|------|------|--------|----------|
| `hid_S` | 空间隐藏维度 | 64 | 32-128 |
| `hid_T` | 时间隐藏维度 | 256 | 128-512 |
| `N_S` | 编码器/解码器层数 | 4 | 3-6 |
| `N_T` | Mid_Xnet 层数 | 8 | 4-12 |
| `groups` | 分组卷积的组数 | 4 | 4-8 |

### 训练参数

| 参数 | 说明 | 默认值 | 推荐值 |
|------|------|--------|--------|
| `epochs` | 训练轮数 | 51 | 1000-2000 |
| `lr` | 学习率 | 0.01 | 0.001-0.01 |
| `batch_size` | 批次大小 | 16 | 8-32 |

## 💡 使用建议

### 1. 训练新模型
```bash
python main.py --epochs 1000 --lr 0.001 --ex_name "MyExperiment"
```

### 2. 调整模型容量
```bash
python main.py --hid_S 128 --hid_T 512 --N_T 12
```

### 3. 在不同数据集上训练
```bash
# Moving MNIST
python main.py --dataname mmnist --in_shape 10 1 64 64

# TaxiBJ
python main.py --dataname taxibj --in_shape 4 2 32 32
```

```bash
python visualize.py
```

## 📚 论文参考

**SimVP: Simpler yet Better Video Prediction**
- 会议: CVPR 2022
- 核心思想: 使用纯 CNN 架构进行视频预测,无需 RNN/Transformer
- 关键创新: Mid_Xnet 在潜在空间进行时间建模

---

## 🎯 核心要点总结

1. **Encoder**: 空间特征提取
2. **Mid_Xnet**: 时间动态建模 (核心创新)
3. **Decoder**: 空间特征重建
4. **Inception**: 多尺度特征融合
5. **端到端训练**: 使用 MSE 损失,简单有效
